# Product Features — AcmeCo Analytics Platform

## Dashboard
- Real-time analytics dashboard with customizable widgets
- Supports line charts, bar charts, pie charts, and tables
- Up to 10 custom dashboards per account
- Data refreshes every 15 minutes (Pro plan) or 1 hour (Free plan)

## Reporting
- Automated weekly and monthly reports via email
- PDF and CSV export formats
- Custom date range filtering
- Report scheduling: daily, weekly, or monthly

## Integrations
- Google Analytics import
- Salesforce CRM sync (Pro plan only)
- Slack notifications for alerts
- Webhook support for custom integrations
- REST API with rate limit of 1000 requests/hour

## Data Management
- CSV import (up to 100MB per file)
- Data retention: 12 months (Free), 36 months (Pro), unlimited (Enterprise)
- GDPR data export and deletion support
- Team-based access controls (Pro and Enterprise plans)

## Limitations
- Maximum 50 users per workspace (Free and Pro)
- Maximum 500 users per workspace (Enterprise)
- No real-time streaming analytics (batch processing only)
- No mobile app (web-only, responsive design)
- No on-premise deployment option
