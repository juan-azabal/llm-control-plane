# Account Management — AcmeCo Analytics Platform

## Account Setup
- Sign up at app.acmeco.com/signup
- Email verification required within 24 hours
- Workspace name must be unique (3-50 characters)

## User Management (Admin only)
- Add users: Settings → Team → Invite Member
- Remove users: Settings → Team → click user → Remove
- Change roles: Admin, Editor, Viewer
- Admin: full access including billing and user management
- Editor: can create and modify dashboards and reports
- Viewer: read-only access to shared dashboards

## Password Management
- Minimum 8 characters, must include uppercase, lowercase, and number
- Password reset: app.acmeco.com/reset-password (link valid for 1 hour)
- Password change: Settings → Security → Change Password
- Two-factor authentication: Settings → Security → Enable 2FA (TOTP only)

## Data & Privacy
- GDPR data export: Settings → Privacy → Export My Data (up to 48 hours)
- Account deletion: Settings → Privacy → Delete Account (irreversible, 30-day grace period)
- Data processing agreement (DPA) available for Enterprise customers

## Plan Changes
- Upgrade: immediate, prorated billing for current period
- Downgrade: takes effect at next billing cycle
- Cancel: access continues until end of current billing period
- Reactivation: within 30 days of cancellation, all data preserved
